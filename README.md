# Kerberos Factory

> [!IMPORTANT]
> **This repository is archived and no longer maintained.**
>
> Kerberos Factory's configuration, deployment manifests and container images are now tracked in the locations listed below. Please use those going forward.

Kerberos Factory is a UI, built for non-technical users, that deploys and configures [Kerberos Agents](https://github.com/kerberos-io/agent) into a Kubernetes cluster. It bypasses the complexity of creating Kubernetes resources by providing a simple UI to connect a Kerberos Agent to your camera and configure it for your use case.
